Приложение для того чтобы узнать текущую погоду для любых координат, также отображает выбранную точку на карте. Пользователь вводит широту и долготу, показывается температура, скорость ветра, время, когда последнйи раз обновлялись данные. Для каждого виджета отображается соответствующая иконка погоды(всего три варианта: солнце, дождь и облачно), а ниже встроенная карта OpenStreetMap с маячком. Можно добавить несколько виджетов и на каждом ввести разные координаты. Для получения информации используется открытое погодное API Open-Meteo. Запросы к нему делаются через fetch и обработку в async/await. Есть оформление для виджетов(отдельные центрированные окошки), кнопок и картинок погоды. Некорректный ввод координат не пропускается, также если не получается получить данные из Open-Meteo выводится ошибка
